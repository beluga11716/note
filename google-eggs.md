**一些Google搜索彩蛋与实用小工具**  

直接在 Google 搜索栏输入对应关键词即可体验（部分可能因地区、浏览器或时间略有差异）。

# Askew
**触发方式**：在 Google 搜索中输入 “askew”。  
**效果**：搜索结果页面会轻微向右倾斜（askew 意为“歪斜的”）。这是一种视觉趣味效果，刷新或搜索其他内容后恢复正常。
![image.png](https://tu.raynard.lol/file/blog/1781868711705.png)
# Mario
**触发方式**：搜索 “Super Mario Bros”、“Mario” 或 “Super Mario”。  
**效果**：右侧知识面板或信息卡会出现一个闪烁的问号砖块。点击它会弹出金币并播放经典音效，致敬经典红白机游戏。
![image.png](https://tu.raynard.lol/file/blog/1781868823969.png)
# Sonic
**触发方式**：搜索 “Sonic”、“Sonic the Hedgehog” 或 “Sonic the Hedgehog game”。  
**效果**：右侧会出现 Sonic 的动画形象。点击 Sonic 他会旋转，点击足够多次（约 25 次）还会变成超级 Sonic。经典复古风格，充满童年回忆。
![image.png](https://tu.raynard.lol/file/blog/1781868864815.png)
# Paddington
**触发方式**：搜索 “Paddington”、“Paddington 2” 或 “Paddington in Peru”。  
**效果**：会出现一个三明治按钮，点击后搜索结果边缘会涂上果酱，并弹出 Paddington 标志性的橘子果酱三明治动画。
![image.png](https://tu.raynard.lol/file/blog/1781874256869.png)
# Minecraft
**触发方式**：搜索 “Minecraft”。  
**效果**：会出现草方块按钮，点击后屏幕角落出现 Steve 的手。你可以点击挖掘方块，收集资源并升级工具。关闭弹出窗口时还可能出现苦力怕，庆祝 Minecraft 周年。
![image.png](https://tu.raynard.lol/file/blog/1781874321434.png)
# 67
**触发方式**：搜索 “67”、“6-7” 或 “six seven”。  
**效果**：页面会短暂上下晃动，模拟网络 meme “67” 的手势动作（上下比划）。有趣的短暂视觉反馈。
![image.png](https://tu.raynard.lol/file/blog/1781873866552.png)
# Pokemon
**触发方式**：在**平板端Google**搜索第一世代宝可梦名称（如 “Pikachu”、“Charmander” 等）。  
**效果**：通常会触发小型互动或动画效果，部分搜索会出现相关知识卡或迷你游戏元素。

# Drop Bear
**触发方式**：搜索 “drop bear”。  
**效果**：会出现一个警告标志，点击后会有“drop bear”（澳大利亚传说中的掉落熊）从天而降的动画，带有幽默警示意味。
![image.png](https://tu.raynard.lol/file/blog/1781873911916.png)
# Georges Seurat
**触发方式**：搜索 “georges seurat”。  
**效果**：知识图谱中会出现绘画图标，点击后搜索结果会切换成点彩派（pointillism）风格，模仿这位画家的艺术特色。

# Breathing Exercise
**触发方式**：搜索 “Breathing Exercise”。  
**效果**：出现一个引导呼吸练习界面（约 1 分钟），有视觉提示跟随吸气和呼气。非常实用的放松工具，可用于减压或会议前后。
![image.png](https://tu.raynard.lol/file/blog/1781873976851.png)
# Cha-Cha Slide
**触发方式**：搜索 “Cha-Cha Slide”。  
**效果**：出现麦克风按钮，点击后浏览器会跟随歌曲歌词“舞动”页面，播放经典舞曲的互动效果（曾因原唱去世短暂下架，现已恢复）。

# Conway's Game of Life
**触发方式**：搜索 “Conway's Game of Life”。  
**效果**：出现可交互的生命游戏模拟器，展示细胞自动机规则的演化过程，适合喜欢数学和模拟的用户。放着不动左右两边会出现效果 演化是自动演化的

# Dinosaur Game
**触发方式**：Chrome 浏览器离线时（或直接访问 chrome://dino），或搜索相关触发。  
**效果**：经典的 Chrome 恐龙跑酷小游戏，按空格或点击跳跃避开仙人掌。支持离线娱乐，是 Google 最著名的隐藏游戏之一。
![image.png](https://tu.raynard.lol/file/blog/1781874075544.png)
# Do a Barrel Roll
**触发方式**：搜索 “Do a barrel roll” 或 “z or r twice”。  
**效果**：整个搜索结果页面旋转 360 度（致敬《星际火狐》）

# Fidget Spinner
**触发方式**：搜索 “Fidget Spinner” 或 “spinner”。  
**效果**：出现可交互的指尖陀螺，可用鼠标或触摸旋转，还可能有数字转盘选项。随时随地解压玩具。

# Flip a Coin
**触发方式**：搜索 “Flip a Coin” 或 “flip a coin”。  
**效果**：Google 直接帮你抛硬币，显示正面或反面结果。实用的小决策工具。

# https://doodles.google/
**触发方式**：直接访问 https://doodles.google/ 或搜索 Google Doodles。  
**效果**：浏览 Google 历年所有 Doodle 作品，许多是可玩的互动游戏和动画合集。

# Is Google Down?
**触发方式**：搜索 “Is Google Down?”。  
**效果**：Google 会幽默地回答 “No.”（目前正常）
![image.png](https://tu.raynard.lol/file/blog/1781874142687.png)
# Metronome
**触发方式**：搜索 “Metronome”。  
**效果**：出现可调节 BPM（40-218）的节拍器工具，适合音乐练习或节奏训练。

# Pac-Man
**触发方式**：搜索 “Pac-Man”。  
**效果**：直接启动经典吃豆人游戏（或相关 Doodle 版本），可在浏览器中玩完整关卡。

# Play Snake
**触发方式**：搜索 “Snake” 或 “Play Snake”。  
**效果**：启动经典贪吃蛇游戏，用方向键控制蛇吃食物成长。Google 提供的怀旧小游戏。

# Wubba Lubba Dub Dub
**触发方式**：搜索 “Wubba Lubba Dub Dub”。  
**效果**：会出现 “Did you mean: I am in great pain please help me.” 的提示，致敬《瑞克和莫蒂》中的梗。对粉丝来说是惊喜彩蛋。
![image.png](https://tu.raynard.lol/file/blog/1781874175711.png)
# What Sound Does a Dog Make?
**触发方式**：搜索 “What sound does a dog make?”（可替换为其他动物，如 cow、pig 等）。  
**效果**：出现动物声音面板，可点击播放对应叫声。适合儿童或趣味学习。

# Timer
**触发方式**：搜索 “Timer” 或 “set a timer for X minutes”。  
**效果**：弹出倒计时工具，可直接设置时间并计时，非常实用的厨房/工作辅助。

# Tic-Tac-Toe
**触发方式**：搜索 “Tic-Tac-Toe” 或 “tic tac toe”。  
**效果**：启动井字棋游戏，可选择对战电脑（不同难度）或好友，支持记分。

# Solitaire
**触发方式**：搜索 “Solitaire”。  
**效果**：启动经典纸牌接龙游戏，支持单翻牌或三翻牌模式，适合打发时间。

# Roll a Die
**触发方式**：搜索 “Roll a Die” 或 “roll dice”。  
**效果**：Google 帮你掷骰子，可选择不同面数，实用且有趣的随机工具。
