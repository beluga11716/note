首先，其实steam的客户端是可以装浏览器插件的。steam的客户端本质上就是一个Chromium内核的浏览器，所以可以装steamdb这种拓展来优化steam的页面。在看游戏时可以看到历史最低，最低价格区域currency lowest recorded price和游戏代码编号，以及一些客户端没有的功能。也可以装翻译蛙这种插件，在社区里浏览国外大佬的作品或者创意工坊时可以翻译成中文。

# **教程**：
## step1 在steam打开一个新标签页

![image.png](https://tu.raynard.lol/file/blog/1780215568338.png)

## step2 复制chrome web store链接
> [!IMPORTANT] IMPORTANT
> 要有外网环境才能访问chrome web store链接 如果没有外网环境 直接看下一步

准备好对应的拓展的链接先 比如**steamdb**就是：

```
https://chromewebstore.google.com/detail/steamdb/kdbmhfkmnlmbkgbabkdealhhbfhlmmon
```

**翻译蛙**就是：
```
https://chromewebstore.google.com/detail/%E9%99%AA%E8%AE%80%E8%9B%99-%E7%BF%BB%E8%AD%AF%E8%88%87%E5%AD%B8%E7%BF%92/modkelfkcfjpgbfmnbnllalkiogfofhb
```
![image.png](https://tu.raynard.lol/file/blog/1780215590726.png)


## step3 在steam装上拓展
在新选项卡打开之后会发现链接是可以选中的 
![image.png](https://tu.raynard.lol/file/blog/1780215619144.png)
把原本的链接删掉 把刚复制的拓展链接粘贴进去 回车就能进到拓展界面 
![image.png](https://tu.raynard.lol/file/blog/1780215643012.png)
直接点击安装就完成了 他会弹窗选择安装位置 **不介意的默认随便选就行** **介意的就提前在steam的根目录下创建个新的文件夹就行** 然后保存就可以使用了
> [!TIP] TIP
> btw 翻译蛙的快捷翻译键是alt+E 在浏览时直接alt加e非常方便的就翻译了
还有就是为什么不推荐沉浸式翻译这个插件 虽然我也用了很长时间 但是听说它有数据泄露的事件所以不做推荐 具体详情可看https://www.bilibili.com/video/BV1aZttzsEC6 这个视频

在写的时候去看了相关的视频 底下评论区也有写泄露这件事 然后沉浸式翻译插件官方有解释 仁者见仁吧
![image.png](https://tu.raynard.lol/file/blog/1780215672448.png)

## 没有外网环境的安装方法
直接下载对应的拓展的文件 即.crx文件
这里可以直接下载我传的
https://wwbaq.lanzouu.com/b009i2wxvc
密码：beluga
然后在steam新选项卡那里输入`chrome://extensions`
打开右上角的开发者模式
然后点加载已解压的拓展程序 选中对应的文件即可安装
![image.png](https://tu.raynard.lol/file/blog/1780215714257.png)
