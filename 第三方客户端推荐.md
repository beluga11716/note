**第三方客户端推荐合集：开源、无广告、高颜值日常App替代方案**

> [!IMPORTANT] IMPORTANT
> **安全提醒**：  
>- 优先使用 **[F-Droid](https://f-droid.org/)** 安装（最安全）。  
>- GitHub Releases 下载APK后，建议验证签名。  
>- 第三方客户端可能因官方API调整暂时不可用，请关注项目更新。  
>- **仅供个人学习交流使用**，尊重版权与服务条款。  

# Bilibili 第三方客户端

**PiliPala**：Flutter开发的Bilibili第三方客户端。  
对应软件：**Bilibili**  
**优点**：界面现代流畅、功能较全（视频、直播、弹幕）。  
**缺点**：偶尔需更新适配官方变化。  
github：  
[`guozhigq/pilipala`](https://github.com/guozhigq/pilipala)  
下载：https://github.com/guozhigq/pilipala/releases

**PiliPlus**：Flutter开发的Bilibili第三方客户端（增强版）。  
对应软件：**Bilibili**  
**优点**：功能丰富、界面现代。  
**缺点**：稳定性依赖维护，可能有小Bug。  
github：  
[`bggRGjQaUbCoE/PiliPlus`](https://github.com/bggRGjQaUbCoE/PiliPlus)  
下载：https://github.com/bggRGjQaUbCoE/PiliPlus/releases

# 网易云音乐 第三方客户端
> [!IMPORTANT] IMPORTANT
> 仅支持电脑端 电脑端macos windows linux三端都有对应软件

**YesPlayMusic**：高颜值网易云第三方播放器（跨平台）。  
对应软件：**网易云音乐**  
**优点**：界面优雅丝滑、MV和歌词支持好、无广告。
**缺点**：依赖非官方API，偶尔失效。  
github：  
[`qier222/YesPlayMusic`](https://github.com/qier222/YesPlayMusic)  
下载：https://github.com/qier222/YesPlayMusic/releases

**simple-cloud-music**：简洁PWA版网易云客户端。  
对应软件：**网易云音乐**  
**优点**：轻量易用、支持歌词。  
**缺点**：功能相对基础，不如原生App完整。  
github：  
[`dufu1991/simple-cloud-music`](https://github.com/dufu1991/simple-cloud-music)

# QQ音乐 第三方客户端
> [!TIP] TIP
> 这个还是直接用官方客户端吧 其实下面这些都不是很好用

**MusicFree**：插件化无广告音乐播放器（支持QQ音乐等）。  
对应软件：**QQ音乐**（及多平台）  
**优点**：高度自定义、Material Design美观、动画流畅、功能齐全。  
**缺点**：插件需单独安装，部分源不稳定。  
github：  
[`maotoumao/MusicFree`](https://github.com/maotoumao/MusicFree)  
下载：https://github.com/maotoumao/MusicFree/releases （强烈推荐）

**QQMusicWeb**：WebView包装的QQ音乐客户端。  
对应软件：**QQ音乐**  
**优点**：去广告、简单易用。  
**缺点**：基于网页，体验不如原生流畅。  
github：  
[`xlzhen-940218/QQMusicWeb`](https://github.com/xlzhen-940218/QQMusicWeb)

# Telegram 第三方客户端

**Forkgram**：Telegram官方增强fork。  
对应软件：**Telegram**  
**优点**：轻量、功能丰富。  
**缺点**：部分增强功能可能有兼容性问题。

**缺点**：无官方登录支持。

github：  
[`TeamNewPipe/NewPipe`](https://github.com/TeamNewPipe/NewPipe)

对应软件：**YouTube**

github：  
[`libre-tube/LibreTube`](https://github.com/libre-tube/LibreTube)

# F-Droid 相关

**F-Droid**：开源应用仓库平台。  
对应软件：**应用商店**  
**优点**：安全、无广告、无追踪。  
**缺点**：应用数量少于Google Play。  
官网：https://f-droid.org/  
下载：https://f-droid.org/ （直接下载APK）

**Droid-ify / Neo Store**：现代Material F-Droid客户端。  
对应软件：**F-Droid客户端**  
**优点**：界面漂亮流畅。  
**缺点**：需额外安装。  
下载：F-Droid搜索 Droid-ify 或 Neo Store

# **总结与建议**：  
以上客户端大多采用 Jetpack Compose 或 Flutter 开发，**颜值和流畅度**普遍优秀，且**无广告**是核心筛选标准。日常强推 **MusicFree**、**PiliPala/PiliPlus** 和 **NewPipe**。安装前检查项目更新，只用官方Releases页面下载。
